# Breakcraft
Breakcraft war ein kleines Minecraft Servernetzwerk, welches eigenständig geführt wurde.
Diese Repository beinhaltet eine Überarbeitung einiger Inhalte des älteren [Repository](https://github.com/adrian-lxm/Breakcraft-Projects), da diese aufgrund meiner damaligen Kenntnisse ineffizienten und teilweise schlechten Code beinhaltete.
## Überarbeitete Plugins

### 1. Lobby Plugin
Zuständig für die Lobby stellt dieses Plugin ein Menu zur Auswahl der Modi bereit inklusive der Möglichkeit auch NPCs zum Schlagen hierfür zu nutzen. Ränge und Spieleranzahl wurden auch vom Proxy abgefragt und dem Spieler als Scoreboard dargestellt.

#### 2. Proxy Plugin
Damals noch ein Bungeecord Plugin wurde dieses von einem erneuerten Velocity Plugin ersetzt, welches ein Ban-System mit Datenbankintegration zu Verfügung stellte, sowie ein PluginMessaging Service um die Spieler mit Hilfe der anderen Plugins zu den richtigen Unterservern zu schicken.

### 3. Survival Plugin
Wie der Name bereits aussagt, ist Survival ein Spielmodi, welches die typische Minecraft Erfahrung um einige Zustzfeatures erweitert hat. Neben einer eigenen Wirtschaft konnten Spieler Grundstücke kaufen und untereinander handeln, riesige Strukturen bauen und sich untereinander verbünden.
